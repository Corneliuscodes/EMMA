## Ease My Mind Africa (EMMA)

This is a static website for **Ease My Mind Africa (EMMA)**, a tech-driven mental health initiative that leverages AI and creative therapy to educate, support, and research mental health in Africa.

### Getting Started

- Open `index.html` in your browser to view the site.
- Make sure the `styles`, `bootstrap`, `fontawesome`, `assets`, and `scripts` folders remain in the same structure as in this project.

### Technologies Used

- HTML, CSS, JavaScript
- Bootstrap
- Font Awesome
